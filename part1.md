## 凡例

為了使讀者快速理解書中的內容，這本書中使用到的凡例如下：


* ```ls()``` 是在行內中 R 的指令或其他的語言指令，本例中為 R 的 ls 函式

* 灰色框起來的部分代表是 R 的指令範例
```R
# This is the comment
# comment line 2
> a <- b <- 1
> a
[1] 1
```
上面以 # 開頭的就是註解，有 > 開頭的就是 R 指令，
其餘是執行後輸出在 R console 的結果
